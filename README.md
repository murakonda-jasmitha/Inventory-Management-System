# Inventory Management System

## 📌 Project Overview
The Inventory Management System is a Python and MySQL-based application developed in Jupyter Notebook. It allows an owner to manage inventory and enables users to purchase products with automatic bill generation.

## 🚀 Features

- Owner Login
- User Registration
- User Login
- Add Products
- View Products
- Update Products
- Delete Products
- Purchase Products
- Bill Generation
- MySQL Database Integration

## 🛠️ Technologies Used

- Python
- MySQL
- Jupyter Notebook
- mysql-connector-python

## 🗄️ Database Tables

- owner
- users
- products
- orders

## ▶️ How to Run

1. Install Python.
2. Install MySQL Server.
3. Create the database.
4. Import the SQL file.
5. Install mysql-connector-python.
6. Open the notebook.
7. Run all cells.
